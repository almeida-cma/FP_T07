📖 O que são Procedimentos?

🔹 Procedimento sem parâmetros

Executa uma tarefa específica sem receber dados externos. Basta chamar seu nome para executar.

🔹 Procedimento com parâmetros

Recebe valores (parâmetros) para trabalhar internamente, permitindo personalizar sua execução.

🔹 Variável Global

Declarada fora da função inicio(). Pode ser acessada e modificada por qualquer procedimento.

🔹 Variável Local

Declarada dentro de um procedimento. Existe apenas durante sua execução e é reiniciada a cada chamada.

📌 Exemplo 1 - Procedimento sem parâmetros

🔹 Executa uma tarefa fixa. Não recebe dados. Basta chamar pelo nome.
programa
{
    // Procedimento sem parâmetros
    funcao vazio saudacao()
    {
        escreva("Olá! Bem-vindo ao programa!\n")
    }
    
    funcao inicio()
    {
        saudacao()  // Chamada do procedimento
    }
}

📌 Exemplo 2 - Procedimento com parâmetros

🔹 Recebe valores (parâmetros) para personalizar sua execução.
programa
{
    // Procedimento com parâmetro
    funcao vazio saudacaoPersonalizada(cadeia nome)
    {
        escreva("Olá, ", nome, "! Seja bem-vindo!\n")
    }
    
    funcao inicio()
    {
        cadeia nomeUsuario
        
        escreva("Digite seu nome: ")
        leia(nomeUsuario)
        
        saudacaoPersonalizada(nomeUsuario)
    }
}

📌 Exemplo 3 - Variável Global

🔹 Variável declarada fora do inicio(). Pode ser alterada por qualquer procedimento.
programa
{
    // Variável global
    inteiro contador = 0
    
    // Procedimento que altera variável global
    funcao vazio incrementar()
    {
        contador = contador + 1
        escreva("Contador agora é: ", contador, "\n")
    }
    
    funcao inicio()
    {
        escreva("Valor inicial: ", contador, "\n")
        incrementar()
        incrementar()
        incrementar()
    }
}

📌 Exemplo 4 - Variável Local

🔹 Variável declarada dentro do procedimento. É reiniciada a cada chamada.
programa
{
    // Procedimento com variável local
    funcao vazio incrementarLocal()
    {
        inteiro contadorLocal = 0
        
        contadorLocal = contadorLocal + 1
        escreva("Valor local: ", contadorLocal, "\n")
    }
    
    funcao inicio()
    {
        incrementarLocal()  // Mostra 1
        incrementarLocal()  // Mostra 1 (sempre reinicia)
        incrementarLocal()  // Mostra 1
    }
}

📌 Exemplo 5 - Sistema Bancário

🔹 Aplicação prática com múltiplos procedimentos e menu interativo.
programa
{
    funcao vazio limparTela()
    {
        limpa()
    }
    
    funcao vazio exibirMenu()
    {
        limparTela()
        escreva("================================\n")
        escreva("      SISTEMA BANCÁRIO         \n")
        escreva("================================\n")
        escreva("1 - SALDO\n")
        escreva("2 - EXTRATO\n")
        escreva("3 - EMPRÉSTIMO\n")
        escreva("0 - SAIR\n")
        escreva("================================\n")
        escreva("Escolha: ")
    }
    
    funcao vazio exibirSaldo()
    {
        limparTela()
        escreva("========== SALDO ==========\n")
        escreva("Saldo atual: R$ 2.547,89\n")
        escreva("\nPressione ENTER...")
        cadeia pausa
        leia(pausa)
    }
    
    funcao vazio exibirExtrato()
    {
        limparTela()
        escreva("========== EXTRATO ==========\n")
        escreva("15/03 - Compra     - R$ 347,50\n")
        escreva("12/03 - Pagamento  - R$ 129,90\n")
        escreva("10/03 - Saque      - R$ 200,00\n")
        escreva("\nPressione ENTER...")
        cadeia pausa
        leia(pausa)
    }
    
    funcao vazio exibirEmprestimo()
    {
        limparTela()
        escreva("========== EMPRÉSTIMO ==========\n")
        escreva("Procure sua agência para liberação.\n")
        escreva("\nPressione ENTER...")
        cadeia pausa
        leia(pausa)
    }
    
    funcao inicio()
    {
        inteiro opcao
        
        faca
        {
            exibirMenu()
            leia(opcao)
            
            se (opcao == 1)
            {
                exibirSaldo()
            }
            senao se (opcao == 2)
            {
                exibirExtrato()
            }
            senao se (opcao == 3)
            {
                exibirEmprestimo()
            }
            senao se (opcao == 0)
            {
                limparTela()
                escreva("Sistema encerrado!\n")
            }
            senao
            {
                escreva("\nOpção inválida! Digite 0-3\n")
                cadeia pausa
                leia(pausa)
            }
        } enquanto (opcao != 0)
    }
}
